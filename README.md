# Calculator

An interactive Advanced Calculator built using HTML, CSS, and JavaScript. This project performs basic and advanced mathematical calculations while providing a clean and user-friendly interface.

## 📌 Project Overview

This project is designed to simulate a modern calculator that supports both standard arithmetic operations and additional mathematical functions.

Users can perform calculations directly using the on-screen buttons or their keyboard.

The application also includes memory operations and expression evaluation for a better user experience.

## 🚀 Features

* Addition, subtraction, multiplication, and division
* Percentage calculations
* Square root calculations
* Exponent operations
* Parentheses support
* Memory functions (MC, MR, M+, M-)
* Clear and delete functionality
* Keyboard support
* Real-time display updates
* Error handling for invalid expressions
* Clean and responsive interface

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JavaScript)

## 🧠 Memory Functions

* **MC** – Clear stored memory
* **MR** – Recall stored memory
* **M+** – Add current value to memory
* **M-** – Subtract current value from memory
