# Calculator

A simple calculator built with HTML, CSS, and JavaScript.  
It supports basic arithmetic operations: Addition, Subtraction, Multiplication, and Division.

---

## Project Structure
.
├── 01_index.html   # Main HTML file
├── 01_styles.css   # Styling for calculator layout
└── 01_app.js       # JavaScript logic

---

## How to Run
1. Clone or download this repository.
2. Open the project folder in VS Code.
3. Open `01_index.html` in your browser (just double-click or use Live Server in VS Code).
4. Start calculating!

---

## Features
- Simple, clean UI
- Supports +, -, ×, ÷
- Clear button to reset calculations
- Error handling (e.g., divide by zero)

---

## Preview
Example:
- Input: 5 + 3  
- Output: 8
